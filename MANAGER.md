- **AI 작업 규칙**: [`CLAUDE.md`](./CLAUDE.md) — 파일 구조, 작업 절차, 절대 금지 사항, 디자인 가이드

- **사이트 기획서**: [`PLAN.md`](./PLAN.md) — 요금표, 시설 정보 원본

```
추측으로 진행하지 말고, 거짓말로 둘러대지 말 것.
```

## 5. 안전 원칙 (AI 에이전트 공통)

AI 에이전트는 다음을 지켜야 합니다.

1. **작업 전 `git pull`** — 다른 사람이 먼저 수정했을 수 있음
2. **수정 전 백업** — `cp index.html index.html.bak_YYYYMMDD`
3. **막히면 멈추고 관리자에게 질문** — 추측 진행 금지
4. **비밀번호·토큰 코드에 하드코딩 금지**
5. **사이트 정체성(농어촌민박·공유별장 컨셉) 유지** — 단, 리디자인은 허용 (정체성은 톤, 디자인은 시각)
6. **배포 후 확인** — push 후 1~2분 뒤 https://yeoyu.poomasi.org 에서 직접 확인
