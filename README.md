## 사이트를 수정하는 두 가지 방법

### 방법 A. Claude Code 웹으로 수정하기 (가장 쉬움, 권장)

브라우저에서 https://claude.ai/code 에 접속해 한국어로 부탁만 하면 됩니다.
Claude는 이 폴더에 있는 `CLAUDE.md` 안내문을 자동으로 읽고 규칙대로 작업합니다.

**예시 부탁말:**

- "메인 화면 큰 사진을 새로 받은 사진으로 바꿔줘. 새 사진은 다운로드 폴더에 hero_new.jpg로 있어."
- "갤러리 두 번째 사진을 바꿔줘."
- "주말 요금을 36만원에서 38만원으로 올려줘."
- "소개글에서 '프라이빗 별장 펜션' 부분을 '프라이빗 공유별장'으로 바꿔줘."
- "예약 문의 전화번호를 새 번호로 바꿔줘."

Claude가 알아서:
1. 어떤 파일을 바꿔야 할지 찾고
2. 백업을 만들고
3. 수정한 다음
4. GitHub에 올리고
5. 1~2분 뒤 사이트가 자동으로 새로 배포되도록 합니다.

> 작업 중 Claude가 무언가를 물어보면 답해주세요.
> 잘 모르는 내용이면 후니님께 먼저 물어보고 답해도 됩니다.

### 방법 B. GitHub 웹 편집기로 직접 수정하기

간단한 글자 한두 줄 정도라면 GitHub 웹사이트에서 바로 고칠 수 있습니다.

1. https://github.com/haeory-cyber/yeoyu-site 접속
2. 수정하고 싶은 파일 클릭 (예: `index.html`)
3. 오른쪽 위 연필 아이콘 클릭
4. 내용 수정
5. 아래로 스크롤해서 "Commit changes" 버튼 클릭
6. 1~2분 뒤 https://yeoyu.poomasi.org 에서 확인

> 사진을 바꿀 때는 같은 파일 이름으로 새 파일을 업로드하면 됩니다.
> (예: `hero.jpg`를 새 사진으로 교체할 때 새 사진 이름도 `hero.jpg`로 맞춰서 올리기)

## 폴더 안에 뭐가 들어 있나요?

```
yeoyu-site/
├── index.html        ← 사이트 본체. 글, 디자인, 동작이 다 여기에 있음
├── images/           ← 사이트에 보이는 사진들
│   ├── hero.jpg              메인 화면 큰 사진
│   ├── interior-living.jpg   거실 사진
│   ├── interior-loft.jpg     다락방 사진
│   └── gallery-01~05.jpg     갤러리에 보이는 사진 5장
├── cards/            ← 사용설명서 카드뉴스 (입실 안내용)
├── PLAN.md           ← 사이트 기획서 (요금표, 시설 정보 원본)
├── README.md         ← 지금 보고 있는 이 파일
└── CLAUDE.md         ← Claude Code 웹이 작업할 때 참고하는 규칙
```

대부분의 수정은 `index.html` 파일과 `images/` 폴더 안에서 일어납니다.

## 자주 묻는 질문

### Q1. 사진을 바꾸고 싶어요. 어떻게 해야 하나요?
A. 가장 쉬운 방법은 Claude Code 웹(claude.ai/code)에 "메인 사진을 바꿔줘" 같이 부탁하는 것입니다.
사진은 미리 1MB 이하로 줄여두면 좋습니다. 줄이는 사이트는 https://squoosh.app 추천합니다.
사진 파일 이름은 원래대로 두는 것이 안전합니다. (예: `hero.jpg`는 그대로 `hero.jpg`)

### Q2. 글자만 살짝 고치고 싶어요.
A. GitHub 웹 편집기(방법 B)가 가장 빠릅니다.
`index.html` 파일을 열고 연필 아이콘으로 수정한 뒤 "Commit changes"만 누르면 됩니다.

### Q3. 요금을 바꾸려는데 어디를 고쳐야 하나요?
A. 요금은 `index.html` 안 세 군데에 들어 있습니다. (표시 글자, 자동 계산 변수 등)
세 곳을 모두 같이 고쳐야 하므로, 이건 Claude Code 웹에 부탁하는 것을 권장합니다.
"주말 요금을 36만원에서 38만원으로 바꿔줘"라고 말하면 알아서 세 곳을 다 찾아 바꿉니다.

### Q4. 잘못 올린 것 같아요. 되돌릴 수 있나요?
A. 네, 거의 모든 수정은 되돌릴 수 있습니다.
- GitHub 레포의 "Commits" 탭에서 이전 버전을 볼 수 있고,
- Claude Code 웹에 "방금 한 수정 되돌려줘"라고 부탁하면 됩니다.
- 정말 막히면 후니님이나 품앗이 지미(AI 실장)에게 도움을 요청하세요.

### Q5. 수정한 게 사이트에 안 보여요.
A. 보통 1~2분이면 반영됩니다. 그래도 안 보이면:
1. 브라우저 새로고침을 강하게 (`Ctrl+Shift+R` 또는 `Cmd+Shift+R`)
2. https://github.com/haeory-cyber/yeoyu-site/actions 에서 배포 상태 확인
3. 그래도 안 되면 도움 요청

## 사이트가 어떻게 자동으로 배포되나요?

이 레포의 `main` 브랜치에 변경사항이 올라가면(=커밋이 push 되면),
Cloudflare Pages가 자동으로 새 버전을 빌드해서 https://yeoyu.poomasi.org 에 올립니다.
보통 1~2분 정도 걸립니다. 별도로 배포 버튼을 누르거나 따로 작업할 일은 없습니다.
